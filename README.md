# Куся — питомец для Codex

В этой папке находится готовый питомец Куся версии v2. В комплект включена исправленная анимация бега влево.

## Состав

- `pet.json` — имя, описание и настройки питомца.
- `spritesheet.webp` — все анимации Куси.

Оба файла обязательны. Не переименовывайте их.

## Установка на Windows

1. Полностью закройте приложение Codex.
2. Нажмите `Win + R`.
3. Вставьте `%USERPROFILE%\.codex\pets` и нажмите Enter.
4. Создайте внутри папку `kusya-keeshond`.
5. Скопируйте в неё файлы `pet.json` и `spritesheet.webp` из этого комплекта.

Должна получиться структура:

```text
%USERPROFILE%\.codex\pets\kusya-keeshond\
├── pet.json
└── spritesheet.webp
```

6. Запустите Codex заново.
7. Откройте выбор питомца и выберите «Куся».

## Обновление существующей Куси

Если Куся уже установлена:

1. Закройте Codex.
2. Откройте `%USERPROFILE%\.codex\pets\kusya-keeshond`.
3. Замените старые `pet.json` и `spritesheet.webp` файлами из этого комплекта.
4. Снова запустите Codex.

## Если питомец не появился

- Убедитесь, что файл называется именно `pet.json`, а не `pet.json.txt`.
- Проверьте, что оба файла находятся непосредственно в папке `kusya-keeshond`, без лишней вложенной папки.
- Полностью перезапустите Codex после копирования.
