# ChatGPT Pets

Каталог пользовательских питомцев для Codex. Каждый питомец хранится в собственной подпапке и содержит полный комплект файлов для установки.

## Питомцы

| Питомец | Папка | Описание |
| --- | --- | --- |
| Куся | [`kusya-keeshond/`](kusya-keeshond/) | Кеесхонд-помощник без аксессуаров; умеет работать, ждать, проверять результаты и смотреть в 16 направлений. |

## Структура папки питомца

В каждой папке должны находиться:

- `pet.json` — имя, описание и версия спрайтов;
- `spritesheet.webp` — анимационный атлас;
- `README.md` — отдельная инструкция по установке и обновлению.

## Установка питомца на Windows

1. Закройте Codex.
2. Откройте папку питомца из этого репозитория.
3. Нажмите `Win + R` и введите `%USERPROFILE%\.codex\pets`.
4. Создайте там папку с тем же именем, что и папка питомца в репозитории.
5. Скопируйте в неё `pet.json` и `spritesheet.webp`.
6. Запустите Codex заново и выберите питомца в списке.

Например, для Куси итоговый путь должен быть:

```text
%USERPROFILE%\.codex\pets\kusya-keeshond\
├── pet.json
└── spritesheet.webp
```

Подробные инструкции для конкретного питомца находятся в его `README.md`.

## Добавление нового питомца

Создайте новую папку с уникальным именем, положите туда обязательные файлы `pet.json`, `spritesheet.webp` и `README.md`, затем добавьте питомца в таблицу выше.
